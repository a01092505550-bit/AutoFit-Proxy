개발 방향

-   OCR은 속도와 인식률 문제로 제외

-   모바일은 검색 성능 중심으로 재설계

현재 검색 방식

-   품번 직접입력
-   조건검색
-   숫자 키패드
-   최근검색
-   즐겨찾기

DB 연동

-   mobile.html에서 iveco_parts_master_unified_sheets1_9.json 직접 읽기
-   JSON 구조 자동 탐색
-   다양한 컬럼명 자동 인식
-   Object 데이터 자동 변환

검색 방식

1.  정확히 일치
2.  앞자리 일치
